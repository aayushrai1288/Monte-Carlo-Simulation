# MAMC Course Assignments - Aayush Rai

This repository contains a structured collection of assignments and solutions for the **MAMC (Markov Chain Monte Carlo)** course or related studies. Each folder is organized by assignment number and includes problem statements (PDFs) and their corresponding implementations (Jupyter Notebooks).

## 🛠️ Tech Stack & Requirements

The solutions are primarily implemented using:
- **Python 3.x**
- **Jupyter Notebook**
- Core Libraries: `numpy`, `matplotlib`, `scipy`, `pandas` (depending on the specific assignment)

## 🚀 Getting Started

To explore the solutions:
1. Clone the repository.
2. Navigate to the specific `assignments/assignment_X` directory.
3. Open the Jupyter Notebook (`.ipynb`) using VS Code, JupyterLab, or Jupyter Notebook.

## 📝 Assignments Overview

| Assignment | Focus Area | Key Topics |
|------------|------------|------------|
| 1 | Basics | Probability foundations and initial simulations. |

| 4 | MCMC Intro | Theoretical grounding for MCMC algorithms. |
| 5 | MCMC Core | Implementation of Metropolis-Hastings or Gibbs Sampling. |
| 6 | Diagnostics | Analyzing chain convergence and mixing. |
| 7 | Final Work | Comprehensive application of MCMC methods. |
